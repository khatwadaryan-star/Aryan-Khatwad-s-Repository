# Project Scope Statement - Personal Introduction Webpage

## Project Purpose
Experience how a well-defined plan can help make the technical work of writing HTML more efficient.
## Project Description
Create a formal scope statement using Markdown and then design a basic personal single-page website using semantic HTML.
## Desired Results
A defined project scope written as a Markdown file.  
A valid semantic HTML file that includes personal information like name, major, goals, hobbies, and GitHub link.  
Improved understanding of how planning can improve technical work.
## Exclusions
No programming languages other than Markdown or HTML.
